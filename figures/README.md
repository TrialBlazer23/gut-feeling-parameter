# figures/

This directory contains plots generated by training scripts, organized by version.

---

## Directory Structure

```
figures/
├── v1_baseline/    # Phase 1 CPU baseline plots
├── v2_scale/       # Phase 2 early iteration plots (deprecated version)
├── v3_phase2/      # Phase 2 GPU-scaled plots
└── v4_phase3/      # Phase 3 LM application plots
```

Each subdirectory corresponds to a training version under `training/`. Figures are generated by the corresponding training script and should not be created or edited manually.

---

## Naming Convention

```
{plot_type}_{config}_{seed}.png
```

Examples:
- `learning_curve_config_d_seed42.png`
- `tsne_hidden_states_config_e_seed7.png`
- `surprise_composition_config_d_seed42.png`

## How to Populate

1. Run the corresponding training script on Colab A100 (see AGENTS.md — How to Run)
2. Download the `plots/` output directory from the Colab session
3. Place the downloaded files in the correct `figures/v{n}/` subdirectory

Do not commit figures generated from truncated or `--quick-test` runs. Figures committed to main should correspond to full training runs with results files present in `results/`.
